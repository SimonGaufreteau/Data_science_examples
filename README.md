# Data_science_examples

## What is this repo ?

Machine learning applied to simple data (school project).

Models used :
- MLP classifier
- DT classifier
- Random Forest (classifier + regressor)
- KNeighbors (classifier + regressor)
- SVC classifier
- Linear regressor
